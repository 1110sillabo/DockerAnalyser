#

The setup.py :
 - install the `docker-analyser` command line tool,
 - clone the code of the repository `DockerLambdaanalyser` into the `${HOME}/DockerAnalyser` folder.
 
```
pyton setup.py install
```

In order to start the analysis  

```
docker-analyser ip
```

```
docker-analyser  up
```

```
docker-analyser ps
```
